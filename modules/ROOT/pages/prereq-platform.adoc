== Supported Environments
The following environments are the only environments supported by Anypoint Platform Private Cloud Edition (Anypoint Platform PCE):

* VMWare bare metal virtualization
* Physical servers
* Amazon Web Services (AWS)

If you are using AWS, perform the steps provided in xref:prereq-aws-terraform.adoc[Provision AWS Prerequisites].
